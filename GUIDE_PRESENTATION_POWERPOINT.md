## ❓ ANTICIPATION DES QUESTIONS

### Questions Techniques Probables

**Q1 : "Pourquoi MongoDB et pas SQL ?"**
- ✅ Flexibilité des schémas
- ✅ Performance sur gros volumes
- ✅ Scalabilité horizontale
- ✅ Standard moderne

**Q2 : "Quel est le coût de formation ?"**
- Inclus dans le budget global
- Formation en cascade (train-the-trainer)
- 26 DPE → 300 Chefs → Milliers enseignants

**Q3 : "Peut-on héberger nationalement ?"**
- ✅ OUI, code exportable
- GitHub + Docker
- Serveurs RDC compatibles
- Migration possible à tout moment

**Q4 : "Quid de la sécurité des données ?"**
- ✅ Chiffrement HTTPS
- ✅ Hachage bcrypt (mots de passe)
- ✅ JWT tokens sécurisés
- ✅ Audit logs complets
- ✅ Backups automatiques

**Q5 : "Conformité avec les lois RDC ?"**
- Protection données personnelles
- Hébergement exportable (souveraineté)
- Audit trail (traçabilité légale)

**Q6 : "Pourquoi $10-14K ?"**
- $600 : Hébergement année 1
- $9-13K : 4 modules critiques (3-4 mois dev)
- Développement déjà réalisé : $0 (donné)

**Q7 : "Y a-t-il des coûts cachés ?"**
- ❌ NON : Prix tout compris
- Hébergement : $50/mois
- Maintenance : Incluse (an 1)
- Support : Inclus (an 1)

**Q8 : "Comparaison avec d'autres solutions ?"**
- Solutions commerciales : $50-100K+
